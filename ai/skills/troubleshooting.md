# Troubleshooting

When blocked:

1. State the error and where it occurs
2. List 2–3 likely causes
3. Try the smallest diagnostic step (logs, isolated repro, test)
4. Apply the smallest fix
5. Re-run verification

If a fix requires a big refactor:

- stop and ask first

Rules:

- Do not write debug scripts. Please create unit tests or integration tests instead.
